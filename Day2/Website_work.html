How a Website Works

1. Browser

The user accesses the website using a browser like Google Chrome, Firefox, or Safari.

2. Server

When the user enters a web address, the browser sends a request to a server, which stores all the files needed to display the website (HTML, CSS, JavaScript).

3. HTML (HyperText Markup Language)

HTML is the language used to create the structure of the webpage. The browser receives content like this:

<h1>Welcome to My Website!</h1>
<p>This is the home page.</p>

4. CSS (Cascading Style Sheets)

CSS is used to style the website, controlling things like fonts, colors, and layout. For example:

body { background-color: lightblue; }
h1 { color: darkblue; }

5. JavaScript

JavaScript adds interactivity to the website, making it dynamic. For example:

alert('Welcome to the site!');

6. Summary

The server sends HTML, CSS, and JavaScript files to the user's browser. The browser then displays the website based on these files.
